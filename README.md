# aozora-tf-idf
 青空文庫の作品（芥川竜之介、太宰治、夏目漱石、宮沢賢治）をtf-idfにし、著者名をラベルにロジスティック回帰でモデルを作り、どのぐらいの精度で著者を当てられるのかを検証

# 環境
- python 3.6
- [Mecab](http://taku910.github.io/mecab)が動く環境
- Mecabとpythonをつなぐモジュール(OSに依存するので割愛)

# 結果
## 87.7 %の精度で当てられる
